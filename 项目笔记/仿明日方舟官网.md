**简单粗暴的实现流程**

- DOM结构：从页面上直接扒下来！

- 静态资源：从页面上直接扒下来！

```tsx
import React from 'react';

const Navigation:React.FC = () => {
    return (
        <React.Fragment>
            <div className="header collapsed">
                <a href="javascript:" className="button-menu collapsed">
                    <span className="button-bar"></span>
                    <span className="button-bar"></span>
                    <span className="button-bar"></span>
                </a>
            </div>
        </React.Fragment>
    )
}

export default Navigation;
```

为了确定是否展开状态，引入 `useState` 保存一个状态值，同时下载第三方包 `classnames` 来动态更改元素的 className

```
npm install classname @types/classnames -s
```

由此可以根据上面分写的HTML内容来编写初步的导航栏 react 代码

```tsx
// src/Components/Header/Navigation/index.tsx
import React,{useState} from 'react';
import classnames from 'classnames';

import NavContent from './NavContent';

const Navigation:React.FC = () => {
    const [isExpanded, switchState] = useState(false);

    // 根据isExpanded的状态更改a标签类名
    const statusClass = classnames({
        'collapsed': !isExpanded,
        'expanded': isExpanded,
    });

    return (
        <React.Fragment>
            <div className={classnames('header',statusClass)}>
                <a href="javascript:" className={classnames('button-menu',statusClass)}>
                    <span className="button-bar"></span>
                    <span className="button-bar"></span>
                    <span className="button-bar"></span>
                </a>
                { isExpanded ? <NavContent/> : null }
            </div>
        </React.Fragment>
    );
}

export default Navigation;
```

**效果样式**

导航图标样式，变换动画 `三 → X`的思路是将三根横线中的第一和第三根都缩小一半长度，然后第一根和第三根旋转分别旋转，这样两根短线形成一根长线，与第二个长度一样，第二根线直接旋转，这样就形成了三根线变成两根的效果

```scss
// src/Components/Header/style.scss

/* 导航栏总容器样式 */
.header {
    position: fixed;
    left: 0;
    right: 0;
    top: 0;
    height: 0;
    z-index: 9990;
    opacity: 0;
    transition: all ease-in-out .3s;
}

.ready .header {
    opacity: 1;
    filter: none;
}


/* a标签容器样式 */
a {
    background-color: transparent;

    &:active,
    &:hover {
        outline: 0;
    }
}

.button-menu {
    width: 2em;
    height: 2em;
    padding: .5em;
    margin: .5em;
    display: block;
    position: relative;
    float: left;

    /* 横线样式 */
    .button-bar {
        position: absolute;
        top: 50%;
        width: 2em;
        height: .2em;
        background-color: #fff;
        text-indent: 101%;
        white-space: nowrap;
        overflow: hidden;
        transition: all ease-in-out .3s;

        /* 未展开的横线样式 */
        &:nth-child(1) {
            left: .5em;
            transform-origin: right center;
            transform: translate(0, -.5em);
        }

        &:nth-child(2) {
            transform: none;
        }

        &:nth-child(3) {
            right: .5em;
            transform-origin: left center;
            transform: translate(0, .5em);
        }
    }

    /* 展开样式 */
    &.expanded {
        .button-bar {
            &:nth-child(1) {
                width: 1em;
                transform: translate(0, 0) rotateZ(45deg);
            }

            &:nth-child(2) {
                transform: rotateZ(-45deg);
            }

            &:nth-child(3) {
                width: 1em;
                transform: translate(0, 0) rotateZ(45deg);
            }
        }
    }
}
```
